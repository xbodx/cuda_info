# info

Small script that shows information about the GPU and CUDA

# venv

in windows
```
python -m venv .cuda_info_env
.\.cuda_info_env\Scripts\activate.bat 
python -m pip install --upgrade pip
```

in linux
```
python -m venv .cuda_info_env
source ./.cuda_info_env/bin/activate
python -m pip install --upgrade pip
```
